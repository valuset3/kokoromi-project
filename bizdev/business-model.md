## 参考プラットフォームのビジネスモデル調査

### 1. ETIC. Beyonders（beyonders.etic.or.jp）

- **運営主体**：NPO法人ETIC.（東京都渋谷区）。複数企業が共同参加する「and Beyond カンパニー」コンソーシアム形式で運営。
- **主な収益源**
  - 参加企業（コンソーシアム参加企業）からの協賛・参画費（金額非公開）
  - 行政・財団からの補助金（事業再構築補助金など）
  - 関連イベント「Beyond カンファレンス」参加費（2日券13,000円 / 1日券5,000〜10,000円）
- **料金体系**
  - 個人参加者：無料
  - プロジェクトオーナー（NPO等）：無料（推定）
  - 企業（コンソーシアム参加）：非公開・問い合わせ対応
  - 参加者への報酬例：プランA 月3万円（週6h）/ プランB 月7万円（週2日）/ プランC 月10万円（週3日）
- **規模感**：参加企業17社前後。プロジェクト数十件規模。
- **ビジネスモデルの特徴**
  - 社会課題解決プロジェクトへの越境参加を、企業の人材育成・SDGs施策として切り出し、企業から協賛費を得る構造
  - 複数企業が共同主催することでコストを分散する「コンソーシアム型」
  - 関連サービス「DRIVEキャリア」は求職者無料・採用企業から掲載・手数料を得る派生サービス

---

### 2. サービスグラント（servicegrant.or.jp）

- **運営主体**：認定NPO法人サービスグラント（2005年設立）。日本のプロボノの草分け的存在。
- **主な収益源**
  - 企業向けプロボノプログラムの提供料（「プロボノリーグ」30社200名超が参加）
  - 行政・財団からの助成金・委託費（40以上の団体と連携）
  - 「ふるさとプロボノ」等の自治体委託プロジェクト
  - 寄付・サポーター会費（認定NPO法人のため税制優遇あり）
- **料金体系**
  - NPO・地域団体（支援を受ける側）：基本無料
  - 企業向け社員研修型プロボノ：非公開・問い合わせ対応
  - プロボノ活動の市場換算額：1プロジェクトあたり約350万円相当と公表
- **規模感**：設立20年超。企業協働実績25社以上、行政・財団連携40団体以上。国内最大規模のプロボノ組織。
- **ビジネスモデルの特徴**
  - 企業のCSR・人材育成予算を活用して社員プロボノプログラムを受注する「B2B型越境学習事業」と、行政委託・助成金の両輪
  - プロボノ活動の「金銭換算値」を可視化することで、企業・寄付者へのインパクト訴求を巧みに行っている
  - GRANTプラットフォームで中間支援組織と連携し、全国展開のネットワーク収益を狙う設計

---

### 3. サンカク（sankak.jp）

- **運営主体**：株式会社リクルート（2014年開始、現在はリクルートに統合）
- **主な収益源**
  - 採用成功報酬（採用1名ごと）
  - 企業向け掲載・プログラム料
  - 「ふるさと副業」マッチング：着手金0円・成立時10万円（税抜）
- **料金体系**
  - 個人（副業者・社会人）：完全無料
  - 企業向けスポットディスカッション：無料
  - 企業向けインターンシップ・採用目的プラン：基本料金30万円〜＋採用成功報酬
  - ふるさと副業マッチング：成立時10万円（税抜）のみ
- **規模感**：登録者数16.1万人（2025年1月時点）
- **ビジネスモデルの特徴**
  - リクルートの巨大会員基盤を活用したマッチングプラットフォーム
  - 「スポットディスカッション（無料体験）」で流入を増やし、採用・副業成立時に課金するフリーミアム的設計
  - 地方企業限定サービスとし、「地方創生」文脈で行政・自治体との連携を図る差別化

### 4. しごと旅行（shigoto-ryokou.com）

- **運営主体**：株式会社仕事旅行社（2011年設立。東京都港区）。有料職業紹介事業許可取得。
- **主な収益源**
  - 個人参加者からの体験参加費（メイン収益）
  - 法人向け越境学習・企業研修「仕事旅行 for Business」（金額非公開）
  - ホスト企業への収益分配：参加費の50%がホスト側／50%が仕事旅行社
- **料金体系**
  - 個人参加費：5,000〜30,800円（体験内容・時間・ホストにより異なる）
    - 例）民間図書館館長体験：5,000円 / スパイスセラピスト：13,200円 / イメージコンサルタント：30,800円〜
  - 法人向け研修：非公開・問い合わせ対応
- **規模感**：170種類以上の職業体験を提供（2025年時点）
- **ビジネスモデルの特徴**
  - 個人とホスト（職業体験提供職場）両側を取り込む「2サイドプラットフォーム」
  - 参加費の50%をホストに還元することでホスト側にも収益インセンティブを設計
  - 個人向け（キャリア探索）と法人向け（越境学習・研修）に同一コンテンツで展開する効率的な設計
  - 「旅行」という言葉でキャリア支援・研修の文脈よりも参加ハードルを下げるブランド設計

## KOKOROMi 収益モデル提案

### 基本方針

> **個人は無料（または超低価格）を維持しながら、企業・団体・行政から収益を得る。**
> 三方よしの設計を「ビジネスとして持続可能にする」ためには、誰がお金を出す合理的な動機を持つかを起点に考える。

---

### フェーズ別 収益ロードマップ（仮）

#### Phase 1（今〜6ヶ月）：実績をつくる。お金は後回し。

| 収益源 | 金額感 | 優先度 |
|---|---|---|
| 行政・財団の助成金・補助金 | 30〜100万円/件 | ★★★ |
| 掲載団体は完全無料 | ¥0 | ー |
| 個人参加者も完全無料 | ¥0 | ー |

まず「1件プロジェクト完走」という実績をつくることが最大の資産。お金を取ろうとすると掲載が集まらない段階。

#### Phase 2（6ヶ月〜1.5年）：B2B収益の芽を育てる

| 収益源 | 金額感 | 解説 |
|---|---|---|
| 企業向け越境学習プログラム | 30〜80万円/社/回 | 社員のプロボノ参加を「研修」として企業に売る（サービスグラント型） |
| 掲載団体からの成功報酬 | 3〜10万円/件完走 | プロジェクト完走後に後払いしてもらう |
| 行政委託・補助金 | 50〜200万円/件 | 地域課題×プロボノは自治体との相性が良い |

企業の「人材育成・CSR予算」に乗れると単価が大きい。「社員5名をKOKOROMiのプロジェクトに参加させる＝50万円」という設計が現実的。

#### Phase 3（1.5年〜）：複数の柱を持つ

| 収益源 | 金額感 | 解説 |
|---|---|---|
| 企業向け越境学習（定期契約） | 月30〜100万円/社 | 年間契約で社員が継続的に参加するプラン |
| 掲載プレミアムプラン | 5〜15万円/月 | 優先露出・コーディネーターサポートつき |
| 個人向けプレミアム（任意） | 月1,000〜3,000円 | マッチング優先・フォローアップ面談など |
| イベント・コミュニティ収益 | 3,000〜10,000円/人 | 「越境した人が集まる場」自体の価値化 |
| コンソーシアム型協賛 | 50〜200万円/社/年 | 複数企業がスポンサーとなりKOKOROMiを支える（Beyonders型） |

---

### 収益規模シミュレーション（仮・Phase 2到達時点）

```
企業向け越境学習プログラム  3社 × 50万円 = 150万円/年
掲載団体 成功報酬          10件 × 5万円 =  50万円/年
行政補助金・委託            1〜2件      =  50〜100万円/年
────────────────────────────────────────
合計（目安）               250〜300万円/年
```

これで1人分の人件費（業務委託）と運営コストをまかなえる水準。
「最初の1件完走 → 実績で企業営業 → 越境学習プログラムとして販売」が最も現実的な道筋。

---

### KOKOROMiが狙うべきブルーオーシャン

サービスグラントやBeyondersは「専門スキル（デザイン・IT等）を持つプロフェッショナル」に特化している。KOKOROMiが差別化できるのは：

> **「スキルよりも熱意・共感を持った、教員・公務員・会社員」という層**
> この層は他サービスが掘れていない。副業不可でもプロボノならOK、という設計がここにしか刺さらない。

この独自性を言語化して企業・自治体に営業することが、中長期の競争優位になる。

### 将来的な法人化・組織形態の選択肢

| 形態 | メリット | デメリット |
|---|---|---|
| 株式会社（現状維持） | 柔軟・意思決定が速い | 助成金・寄付の受け取りに制限あり |
| 一般社団法人 | 助成金が取りやすい・信頼感 | 利益分配に制約 |
| 認定NPO法人 | 寄付に税制優遇・行政委託が取りやすい | 認定に時間・手間がかかる |

初期は株式会社のまま助成金を活用し、実績が積まれたタイミングで一般社団法人または認定NPO法人への移行を検討するのが現実的。

## GRANTとの差別化

最終更新: 2026-05-24

### GRANTとは

認定NPO法人サービスグラントが運営する「社会参加プラットフォーム」。全国87のコーディネーター団体（中間支援組織）がNPO・地域団体の登録・進行管理を担う構造が核心。登録参加者2,678名・掲載団体961団体・マッチング実績2,840件（2025年時点）。2022年グッドデザイン賞受賞。

### GRANTとKOKOROMiの違い

| 軸 | GRANT | KOKOROMi |
|---|---|---|
| 参加者像 | 専門スキル（マーケティング・デザイン・IT等）を持つプロフェッショナル | 熱意と共感で動く人。教員・公務員・会社員。副業不可でも参加OK |
| 参加の動機 | 「スキルを社会に還元したい」 | 「このままでいいのか」外の世界を知りたい・変わりたい |
| 期間・コミット | 2〜3ヶ月のプロジェクト型・重め | 1日〜数週間の小さな越境から始められる |
| 何のための場か | 団体の課題解決・成果物を届ける（団体の変化が主目的） | 団体の課題解決 × 参加者の変化（一挙両得・三方よし） |
| コーディネート | 全国87の中間支援組織に委託（プラットフォーム提供型） | KOKOROMiが直接伴走（コミュニティ型） |

### 一言で言うと

> GRANTは「スキルを持つプロが団体を助けるプラットフォーム」。  
> KOKOROMiは「何者かになりたい大人が越境し、自分を発見する場」。

GRANTが救えない層（副業不可・スキルより熱意・宙ぶらりん状態）にKOKOROMiが刺さる。競合というより補完関係に近い。

## 劣化ソリューション分析

最終更新: 2026-05-25

### 「悩める個人」が今やっていること

KOKOROMiのターゲット（三谷綾香タイプ）が、理想の解決策がないから仕方なくやっていること。

#### タイプA：お金を払っていない層（メイン）

| 劣化ソリューション | 実態 | なぜ劣化か |
|---|---|---|
| YouTube自己啓発動画を見る | 「そうだよな」と思うだけ | 行動に繋がらない。消費で終わる |
| SNSのキャリア系投稿にいいねする | 見るだけ。自分は動けない | 眺めるだけで現実は変わらない |
| 転職サイトを何となく眺める | 転職したいわけじゃないのに開く | ミスマッチ。欲しいものが見つからない |
| 自己啓発本を読む | 読んで満足する | 行動に繋がらない |
| 友達に愚痴る | 「わかるわかる」で終わる | 解決しない。停滞を強化 |
| 習い事・資格勉強を始める | 現状維持の慰めになる | 「外の世界」には出られない |

#### タイプB：お金を払っている層

| 劣化ソリューション | 費用 | なぜ劣化か |
|---|---|---|
| コーチング | 月3〜10万円 | 高い・対話だけで社会に出られない・続かない |
| 転職エージェント | 無料（企業側課金） | 転職が前提・宙ぶらりん層には合わない |
| POOLO・スクール | 15〜30万円/コース | 高い・テーマ限定・副業不可層は行動できない |
| あそと base | 月会費（推定） | まだ小さい・社会課題との接続が薄い |
| しごと旅行 | 5,000〜3万/回 | 1回で終わる・深い関係性・継続性がない |

### 最大の劣化ソリューション

> **「YouTube自己啓発動画を見ること」**

タイプAの圧倒的多数がここにいる（三谷綾香の情報源もYouTubeと明記）。お金を払っていないため、KOKOROMiへの移行コストが「初めてお金を出す」決断になる。

KOKOROMiのポジション：「動画を見て終わりにしない。やってみることで、自分が分かる。」

## しごと旅行との比較・差別化

最終更新: 2026-05-25

### しごと旅行の参加者層

- **年齢**：学生〜60代まで幅広い
- **主な動機**：「好奇心で覗いてみたい」が多数。その職業に就きたいは少数派
- **行動特性**：一人参加が不安。初心者でも大丈夫か不安。深い関与は求めていない
- **一部の層**：転職検討中・企業研修での参加（NEC等）・リピーター（10回以上）

### しごと旅行が広がらなかった構造的理由

| 問題 | 実態 |
|---|---|
| 体験が目的化している | 「調香師を覗く」→ それ自体で完結。次がない |
| リピートする理由がない | 1回行ったら「行った」になる |
| ホストのインセンティブが弱い | 参加費の50%がホストに入るが、数千円では積極的に動かない |
| コミュニティがない | 参加者同士がつながらない。口コミが生まれにくい |
| 「面白い仕事」特化で層が狭い | 調香師・選書家・珈琲職人など職人系に偏り、刺さる層が限られる |

→ しごと旅行は「体験コンテンツ販売業」で止まっている。体験の先がない。

### KOKOROMiとしごと旅行の違い

| 軸 | しごと旅行 | KOKOROMi |
|---|---|---|
| 参加者の動機 | 「面白そう」好奇心ドリブン | 「このままじゃまずい」焦燥感ドリブン |
| 受入先のタイプ | 調香師・職人・クリエイター系 | NPO・社会起業家・地域団体（社会課題系） |
| 参加者が得るもの | 「こんな仕事があるんだ」という知識 | 「自分はこういう人間かもしれない」という気づき＋出会い |
| 体験後の動線 | 終わり。次がない | コミュニティへ続く。越境仲間と繋がり続ける |
| リピート設計 | なし | コミュニティ月会費で継続 |
| ホストのメリット | 参加費の50%（数千円） | 理念共感者・ファン・外の視点が来る |
| WHY | 「職業を旅する」（体験的） | 「子どもに夢を語れる大人を増やす」（社会的） |

### 「何屋さん？」

> しごと旅行 ＝ 職業体験ツアー屋さん（1回の体験で完結）  
> KOKOROMi ＝ **悩める社会人が、社会と初めてつながる場をつくる屋さん**

体験は入り口に過ぎない。体験→自己理解→コミュニティという動線全体を持つことが差別化。しごと旅行との最大の違いは「1回で終わるか、続くか」。

## MVP 最初の1件の運営フロー

最終更新: 2026-05-25

```
STEP 1  受入先を開拓する（無料）
        企業・カフェ・農家・NPOなど
        「来てくれるなら歓迎」という場所をゼロから探す
        ↓
STEP 2  参加者を募集する
        SNS・LINE・口コミで告知
        「こんな場所で越境体験できます」
        ↓
STEP 3  体験する
        1日〜数週間、受入先で実際に関わる
        ↓
STEP 4  振り返りインタビュー（ここがコアバリュー）
        体験後にあのっちが参加者と話す
        「どうでしたか？何を感じましたか？」
        体験を言語化し、自己理解を引き出す
```

### この設計のポイント

- **受入先は無料** → 声をかけやすい。「人が来てくれる」だけでメリットがある
- **振り返りインタビュー** → コーチングとの最大の違い。コーチングは最初に話す、KOKOROMiは体験してから話す。体験後だから出てくる言葉がある
- **事前設計が重要** → 受入先に「こういう関わり方をお願いします」と伝える。参加者にもオリエンテーションで期待値を揃える

### 注意点

- 受入先が「何をしてもらえばいいか分からない」となりやすい → 事前にKOKOROMiが体験プログラムを設計してあげる
- 参加者が「何をしに行けばいいか分からない」となりやすい → 事前オリエンテーション必須

## キッザニアから学ぶビジネスモデルの示唆

最終更新: 2026-05-25

### キッザニアの収益構造

```
① 入場料（個人）    子ども：3,200〜4,900円 / 大人：1,900円
② スポンサー料（企業）体験パビリオンを企業が出展。ロッテ・日産・ANA等が出展費を払う
③ 専用通貨「キッゾ」 稼ぐ→使う→消費の循環設計
```

個人と企業の両方からお金を取る「両面市場」。体験コンテンツの制作費を企業に払わせ、個人からも入場料を取る。企業は「子どもに自社を体験させる広告費」として払う。

### KOKOROMiへの応用

> 受入団体（企業・NPO）は「理念共感者が来てくれる広告費・採用費」として払う  
> 個人は「越境体験・自己理解」への投資として払う

両方から取れる設計が将来的に成立する。最初は「個人だけ」から始め、実績ができれば「団体にも課金」という順番で育てる。
