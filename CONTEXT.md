## コアコンセプト（最新版）

> **「面白い選択をした人に、会いに行く」**

- 職業（What）ではなく**生き方・選択（Why）**にフォーカス
- 仕事はあくまで手段。目的は「こういう生き方があるんだ」という出会い
- **表の目的**: その人のWHYを聞く・生き方に触れる
- **裏の目的**: その人が属する事業・組織の現場を体験する（1対1ではない）
- ホスト選定基準: **「なぜやっているか」を語れるか**（職業の種類ではない）

### 避けること
- ✕ スキル習得だけで終わる体験
- ✕ 労働力としての仕事体験・募集

### 体験の3軸
1. **外に出る経験** — 日常の外側に踏み出す
2. **生き方・ストーリーを聞く** — 「なぜ」の言葉が価値観を揺さぶる
3. **自分が挑戦する機会** — 見るだけで終わらず、関わる・つくる・やり遂げる

## ビジネスモデル（最新版）

### Model B（個人課金）— NOW・Phase 1優先

- 参加者が体験プロジェクトの参加費を払う
- 想定参加費: **1,500〜3,000円/回**（将来は5,000円も検討）
- KOKOROMiの取り分: 50%程度

**将来形：月額コミュニティ（3,000円/月）**
- メンバーが月1回以上の体験イベントに参加できる
- 参加者コミュニティへのアクセス（同じ価値観の仲間）
- 過去の体験コンテンツが見放題
- 100人達成で月30万円の継続収益

### Model A（ホスト・企業課金）— Phase 2以降

- 受入先（団体・組織）がPR費・採用広報費として払う
- 参照モデル: **しごとリーチ！（ビズリーチ）**
  - 企業が「自社をドキュメンタリー化してもらう」ために費用を払う
  - 個人参加者は安く体験できる → 集客しやすい
- 実績が出てから（「KOKOROMiに出ると認知が広がる」と言えるようになってから）

### Phase 3以降: 法人研修

- 「社員の視野を広げる越境体験プログラム」として企業に売る
- 1回 30〜100万円

### 競合整理

| 競合 | 差別化 |
|------|--------|
| 仕事旅行 | What（職業体験）vs **Why（生き方）** |
| 海外旅行 | 価値観を変えに行く目的は同じ。「わざわざ海外に行かなくても国内で同じ体験ができる」 |
| コーチング | 月3〜10万円 vs 月3,000円。体験型で価値観が揺れる |

## 俗人性問題と解決策

**問題**: ホスト（個人）が離れたら体験が消える

**解決策**:
1. **個人ではなく「場・組織」との継続関係を持つ** → 人が変わっても場は続く
2. **体験をコンテンツ化（記事・動画）** → 1回の体験が何千人にも届く。しごとリーチ！モデル
3. **参加者→ホスト化サイクル** → 体験した人が「次のホスト」になっていく

**具体的な設計**:
```
①KOKOROMiがホストに密着・取材（半日）
    ↓
②note記事＋短い動画を制作・発信
    ↓
③「実際に会いに行きたい」人が体験に申し込む
    ↓
④体験は季節ごとに1回固定日程（年4日以下の負担）
```

⑥ 体験実施
