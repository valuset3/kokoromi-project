## ターゲットペルソナ（アーリーアダプター確定）

### 👩 みほ（27歳・中学校教員）

| 軸 | 内容 |
|---|---|
| **基本属性** | 27歳・女性・地方都市在住・実家暮らし・手取り24万円 |
| **職業** | 中学校の国語教員3年目。担任も部活も持っている |
| **価値観** | 子どもと向き合う仕事は好き。でも「学校の外を知らない自分が、子どもの未来を語っていいのか」という違和感がある |
| **日常** | 7時登校・19時退勤。土日も部活。自分のための時間がほぼない。夜中にSNSで「転職した先生」の記事を読む |
| **情報源** | Instagram・note・X。教師コミュニティのオンラインサロンが気になっている |
| **サービス受容性** | 中。月1,000〜2,000円なら試せる。「辞めなくていい」「副業じゃない」なら安心して動ける |

**朝のルーティン：** 6時半起床。身支度しながらInstagramを流し見。「今日もとりあえず乗り越えよう」と思いながら家を出る

**不満の口癖：**
> 「私って学校のことしか知らないまま、子どもに何を語れるんだろう」

**一番の痛み：**
> 「何かしたいのに、どこにも行けない」という積み重なった無力感

**みほさんの不格好な代替手段（上位3つ）：**
- 転職エージェントに「話だけ聞きに」行く → 精神コスト9。転職するつもりないのに罪悪感
- 外部勉強会を調べるが土日の部活で行けない → 「また行けなかった」自己嫌悪
- コーチングスクールを調べるが「高すぎる」で閉じる → お金コスト8、精神コスト7

**KOKOROMiに出会ったとき：**
> 「これなら、今の仕事を続けながらでもできる。しかも転職しなくてもいいんだ」

**アーリーアダプターに選んだ理由：**
1. あのっちの原体験（元教員）と完全に重なる → 本人が「かつての自分」として語れる
2. 副業できない制約がKOKOROMiの設計と一致する
3. 「子どもに何を語れるのか」という問いがKOKOROMiのビジョンに直結する

**みほさんが動く4条件：**
「転職しなくていい」「副業じゃない」「忙しくても参加できる」「やっても意味がなかった、にならない」

## プロジェクト概要

**KOKOROMi** — 子どもに夢を語れる大人を、もっと増やしたい。

- 30代前後の社会人がプロボノ形式で社会課題に取り組むマッチングプラットフォーム
- 企業・団体を見つける軸（軸A）と参加者を集める軸（軸B）を同時進行
- 目標: 3〜6ヶ月以内にプロジェクト完了1件
