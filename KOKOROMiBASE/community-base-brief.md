## なぜやるか（浅野より）

普段、働いていると同じような環境・価値観の人としか出会わない。でも世の中には、自分では想像できないような生き方をして、全然違うところに幸せを見つけている人がたくさんいる。

そういう人の生き方や働き方に触れることで、自分の価値観が揺さぶられたり、自分の可能性に気づいたり、一歩踏み出すきっかけになってほしい。

そのためにまず、オンラインで週に1回、ゆるく集まるだけの場をつくる。

## 参加フロー

```
X・Instagram・note・口コミ
　↓
LINEオープンチャットへ誘導（参加者同士の交流・認知）
　↓
Zoom集まりに参加したい人は公式LINEへの誘導
　↓
公式LINE内で申し込み（LINE公式アカウントの申し込みフォーム機能を使用）
　↓
申し込み確認 → ZoomのURLを公式LINEで送付
　↓
当日参加
```

**オープンチャット（コミュニティ）**
- 誰でも直接入れる
- 入室時に以下の2問に答えてもらう
  1. 年齢を教えてください（学生も歓迎）
  2. 現在の状況を教えてください（社会人 / 学生 / 自営業 / 無職）

**Zoom集まりへの参加**
- 参加には**公式LINE登録 ＋ 事前申し込み必須**
- 申し込みはLINE公式アカウント内のフォーム機能で完結させる（LINE外に飛ばさない）
- ※設定・動作確認はゆいさんにお願いします。もしLINE内で難しければGoogleフォームに切り替えてOK（浅野に相談）
- **顔出し必須**
- 発言は無理にしなくてOK。ただし申し込んだ責任は持ってもらう
- **申し込んだのに無断で来なかった場合、次回以降の参加をお断りすることがあります**（申し込み案内文に明記する）
- 遅刻・早退は可能。事前の申込時に確認する
- 録画はしない

## 定期集まり

| 項目 | 内容 |
|---|---|
| **頻度** | 毎週 |
| **曜日・時間** | 土曜 21:00〜（仮） |
| **時間** | 60分（終了後、残りたい人は残ってOK） |
| **場所** | Zoom |
| **顔出し** | 必須 |
| **形式** | 座談会（ゲスト1人を招いて、みんなで語り合う） |
| **大切にすること** | 対話。講義・セミナー形式にしない |

**当日のタイムテーブル（目安）**

| 時間     | 内容                                   |
| ------ | ------------------------------------ |
| 0〜5分   | 趣旨説明                                 |
| 5〜15分  | ゲスト自己紹介（10分以内。短くてもOK）                |
| 15〜55分 | 質問・対話タイム<br>（ブレイクアウトルームで分けるのはありかな？？） |
| 55〜60分 | アンケート・写真撮影                           |
| 60分〜   | 自由解散（残りたい人はそのまま残ってOK）                |

**ゲストへのお願い（任意）**
- 自分がどんな人かわかるスライド1枚を用意してもらえると助かる
- 図解・プロフィール的なもので十分。強く求めなくてOK

**事前準備**
- ゲストのプロフィール・紹介文をオープンチャット内で事前に共有しておく

> **このフォーマットは固定ではありません。**  
> やってみながら修正していきます。「こう変えた方がよさそう」と思ったら浅野に共有してください。

- イベント中・イベント後・LINEチャット内など、自然なタイミングでOK

## ゆいさんへの裁量範囲

**任せること**
- オープンチャットの開設・管理・運営
- 毎週集まりの進行・ファシリテーション
- 参加者への日常的な声がけ・コミュニケーション
- 体験への誘導タイミングの判断

**浅野に確認してほしいこと（最初のうち）**

- 運営方針の大きな変更
- 迷ったこと全般（慣れてきたら裁量を広げていく）

> **浅野より：** 初めての試みなので、最初は「これどうしますか？」と気軽に聞いてください。基本は任せます。

## 参考：MyStory YourShift との違い

KOKOROMi BASEは、浅野がこれまで運営してきた「MyStory YourShift」（noteマガジン・Zoom会）の進化版にあたります。

|          | MyStory YourShift | KOKOROMi BASE |
| -------- | ----------------- | ------------- |
| ゲスト話す時間  | 約30分              | 約10分          |
| メインコンテンツ | ゲストのストーリー共有       | 参加者同士の対話      |
| 雰囲気      | インタビュー・登壇に近い      | 輪になって話す・サロン感  |
**大事にしたいこと：** ゲストは「きっかけ」。主役は参加者同士の会話。

## この活動のロードマップ

**フェーズ1（今）：オープンチャット・公式LINEに人を集める**
KOKOROMi BASEを通じて、まずオープンチャット・公式LINEの登録者を増やす。

**フェーズ2：体験への認知・参加者を増やす**
LINEに集まった人に、KOKOROMiの体験（現地フィールドワーク）を知ってもらい、参加者・ホストを増やしていく。

**フェーズ3：有料コミュニティ化**
コミュニティが育ってきたら、週1Zoom集まりを有料化する。有料会員のみが参加できる設計に移行する。

> 今は無料でやる。育ってきたら有料にする。段階を踏む。
