# KOKOROMi SNS 運用戦略

最終更新: 2026-05-27

---

## 全体の流入設計

```
X / Instagram（認知・拡散）
　↓ LINEに誘導（CTAはすべてLINE登録）
KOKOROMi 公式 LINE 登録
　↓ キーワード送信（例：「ピカイチ」）
自動返信：① note詳細記事リンク　② 申込フォームリンク
　↓ 「参加したい」送信
浅野との面談（10〜15分）
　↓
ホスト紹介・日程調整
　↓
体験当日
```

**note の位置づけ**：X/Instagramからの直接リンク先ではなく、LINE登録後の自動返信の中で渡す「詳細資料」。note 自体にも LINE 誘導の CTA を置き、note から流入した人も LINE に集める。

各プラットフォームの役割：
| プラットフォーム | 役割 | CTA |
|---|---|---|
| X | 認知・拡散・共感 | LINE 登録へ誘導 |
| Instagram | ビジュアル訴求・世界観 | LINE 登録へ誘導 |
| LINE | 申込の入口・関係構築 | note 詳細 + 申込フォームを送る |
| note | 詳細コンテンツ（LINE経由で読まれる） | LINE 登録へ誘導 |

## X（Twitter）運用方針

### ポジション
「面白い選択をした大人の話」を届けるアカウント。自己啓発でも求人でもない、「出会い」の入口。

### 投稿タイプと頻度

| タイプ | 内容 | 頻度 |
|---|---|---|
| **体験告知** | 新しい体験プロジェクトの募集開始 | 開始時・締切前に複数回 |
| **ホスト紹介** | ホストの選択・ストーリーを1〜2文で | 週1〜2回 |
| **参加者の声** | 体験後の感想・変化（許可を得て） | 体験後 |
| **日常の問い** | 「今の仕事、誰かの背中を見て選んだ？」など | 週1〜2回 |
| **裏側の発信** | 立ち上げの葛藤・失敗・学び（浅野の言葉） | 週1〜2回 |

## Instagram 運用方針

### ポジション
写真と短い言葉で「その人の選択の空気感」を届ける。X より世界観寄り。

### 投稿タイプ

#### フィード投稿
- **ホスト写真 + 一言引用**：ホストの印象的な言葉を大きく入れた画像
- **体験の場所写真**：ピカイチの空間、子どもたちの様子（許可済みのもの）
- **告知カード**：体験プロジェクトの概要を視覚的にまとめた1枚

#### ストーリーズ
- 体験当日のリアルタイム発信（写真・動画）
- 「参加したい人はここから」と LINE へのスワイプリンク
- アンケート・質問スタンプ（「こんな体験、気になる？」）

#### リール（将来的に）
- 体験ダイジェスト動画（30秒〜1分）
- ホストへの1問1答

### 告知投稿テンプレート（キャプション）

visions の Instagram 投稿（2026-05-27 スクリーンショット確認）を参考にした構成。

```
藤森光一さんがつくった学童に、入ってみませんか。

火曜・金曜 9:30〜15:00 ＠東京都小金井市
===

保育士として長年働き、施設長・統括まで経験した。
それでも「自分の信じる保育」は、その枠の中ではできなかった。

だから辞めた。
クラウドファンディングで108万円を集めて、自分でつくった。

KOKOROMiの体験プロジェクト第1弾です。

▼KOKOROMiとは🌱

「その人がどう生きているか」を、現場で肌で感じる体験です。

●長年の保育現場を辞めた理由を、本人から直接聞ける
●主体性を育てる現場を、空気感ごと体で感じる

```

### 参考：visions の Instagram 投稿のエッセンス（2026-05-27）

**画像の作り方：**
- ホストの人物写真を大きく・インパクト重視
- ホスト名をドーンと大きく表示（「〇〇さんが企む、〜」）
- 日程・時間・場所を画像内に明記
- ブランドロゴを左上に
- 背景は単色でコントラスト強め

**キャプションの構成：**
1. ホスト名 + キャッチコピー（1行）
2. 日程・形式（短く）
3. `===` で区切り
4. 大きい文脈の説明（このイベントの位置づけ）
5. WHY（なぜこの場が必要か・課題感）
6. ＜POINT＞（具体的に何が得られるか）
7. ＜参加のながれ＞（LINE誘導）
8. `**********` で区切り
9. 詳細リンク（プロフィールのLINEリンクへ）
10. ハッシュタグ

### プロフィールリンク運用
- リンクは常に **LINE登録リンク（https://lin.ee/RzAsEhv）** に向ける
- 複数の体験が出てきたら linktree 経由で各 LINE 誘導ページに分岐させる

## LINE 運用方針

### ポジション
X/Instagram で興味を持った人を「クローズドな関係」に引き込む場所。
先行案内や個別対応はすべてここで行う。

### キーワード応答設計

| キーワード | 返信内容 |
|---|---|
| `ピカイチ` | 藤森さんの体験詳細 + note リンク + 「参加したい」と送ってと案内 |
| `参加したい` | 浅野から手動返信（日程調整 or 簡単な確認） |
| （未登録キーワード） | 「何か気になることはありますか？」と手動対応 |

### 自動返信メッセージ（「ピカイチ」）

```
〇〇さん、

「ピカイチ」の体験プロジェクトに興味を持っていただき、
ありがとうございます！

保育士を辞めて自分の学童をつくった藤森光一さんの現場に、
一緒に入りに行きましょう🌱

▼体験の詳細はこちら（note）
https://note.com/kokoromi_jp/n/（記事URL）

▼参加申し込みはこちら
（Googleフォーム URL）

参加を希望される方は、申し込みフォームからお送りください。
確認後、代表の浅野から折り返しご連絡します。
```

### LINE 限定コンテンツ（将来）
- 次の体験プロジェクトの先行案内
- ホストのインタビュー記事（note 公開前）
- 参加者コミュニティへの招待

### 実装メモ
- **キーワード自動応答**：LINE公式アカウントマネージャーの「キーワード応答」機能で設定
- **現状**：まず手動対応でスタート。参加者が増えてきたら自動応答を整備

### 参考：visions の自動返信構成（2026-05-27 スクリーンショット確認）
```
① 名前付きの挨拶（「〇〇さん、」）
② イベント名 + 感謝の一言
③ 体験のエッセンスを一言で
④ ▼イベント詳細URL
⑤ ▼参加申し込みURL
⑥ リッチメニュー（カード型：詳細ページ + 申し込みページ）
```

## 連動スケジュール（藤森さんの体験 初回募集）

写真が届き次第、以下のスケジュールで動く。

| タイミング | X | Instagram | LINE |
|---|---|---|---|
| **公開日** | 告知投稿（LINE登録へ誘導） | フィード投稿（LINE登録へ誘導） | 登録者への一斉案内（note + フォームを送付） |
| **公開2〜3日後** | ホスト紹介投稿（LINE誘導） | ストーリーズ（LINE リンクスタンプ） | — |
| **締切1週間前** | リマインド（残り◯名 + LINE誘導） | ストーリーズ（「もうすぐ締め切り」） | — |
| **体験当日** | 当日の様子（許可があれば） | ストーリーズ（現場の空気感） | — |
| **体験後** | 参加者の感想（許可を得て） | フィード（体験レポート写真） | 次の体験の先行案内 |
