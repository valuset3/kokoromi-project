# KOKOROMi 運営メンバー — 行動指針（Value）

ミッションの実現やビジョンへの到達を、どう最短で進めるかを共有するための価値観です。

## バリュー（価値観）

### 1｜日々の変化を受け入れる

人の考えも事業の方針も日々変わり続けている。  
「今までは◯◯だったから」「前は◯◯と言っていたのに」で止まらず、変化に敏感でいることを大切にする。

### 2｜問い→提案→実行まで一気通貫

違和感・興味・気づきは言語化して「問い」にする。  
指摘で終わらせず、提案して実行までやり切る。  
完璧を待たず、気づいた瞬間にまず動く。

### 3｜やり取りはオープンに

仕事に関わる相談や決定は、閉じたDMではなく、みんなが見える場で共有する。  
透明性を大切にし、誰が見ても理解できるコミュニケーションを心がける。

### 4｜越境と挑戦を自ら体現する

KOKOROMiは”挑戦する大人の背中”をつくる場所。  
運営メンバーが挑戦していなければ説得力はない。  
昨日の自分よりも1つでもレベルアップできるように、小さくても自分自身の成長を止めない。
