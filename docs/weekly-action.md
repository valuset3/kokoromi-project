**目標：合計30件に声をかける**

| 時期 | 目標件数 | アクション |

| 6月中 | 10件 | QWS・POOLO・SHIMOKIT ACOLLEGE周辺から優先的に打診 |

| 8月末 | 30件（累計） | 幅を広げてアプローチ |

**ホスト選定基準：** 職業の種類ではなく「なぜやっているかを語れるか」

**1件決まった後の流れ：**
```
①口頭OK
    ↓
②ヒアリングコール（30〜45分）
    ↓
③体験プログラム設計（タイムライン・日程・人数）
    ↓
④参加者とマッチングして実施
    ↓
⑤振り返り・フィードバック収集
    ↓
⑥コンテンツ化（サイト掲載・SNS発信）
    ↓
⑦実績を見せながら次の打診へ
```

## 会計・事業計画

補助金申請など公的な資金獲得に向けて、正式な事業計画書と収支計画を整える。

**前提：法人登記が完了してから本格着手**

| 時期 | タスク |
|---|---|
| 法人登記完了後すぐ | 法人口座開設 |
| 7月末 | 年間収支計画・事業計画書の初稿作成 |
| 8月 | 申請対象の補助金・助成金を調査・絞り込み |
| 9月以降 | 補助金申請 |

**作成が必要なもの：**
- [ ] 年間収支計画（売上・費用・利益の見通し）
- [ ] 事業計画書（事業概要・市場・収益モデル・スケジュール）
- [ ] 資金計画（自己資金・補助金・将来の調達見通し）

**参考：相談済みの窓口**
- 東京創業ステーション（2026-05-27相談済み）← 補助金情報も持っているので再訪する価値あり
